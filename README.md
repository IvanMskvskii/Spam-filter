# Spam-filter
Целью проекта было создание алгоритма автоматитческой классификации SMS-сообщений с целью выявления спама.
Данная задача была решена с помощью построения модели, способной отличать спам-сообщения от обычных на основе их текстового содержания и других признаков.

В качестве данных используется датасет SMS Spam Collection Dataset с платформы Kaggle: https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset

В качестве целевой метрики был выбран F1 score, так как она хорошо работает при задачах с дисбалансом классов.
В ходе работы были выполнены очистка и предобработка текстовых данных, токенизация и преобразование текста в числовое представление. Для обучения моделей использовались различные методы векторизации текста (CountVectorizer, TfidfVectorizer), а также алгоритмы машинного обучения, включая логистическую регрессию и различные варианты наивных байесовских классификаторов.

В рамках проекта были обучены и сравнены несколько моделей классификации, проведена оценка их качества и проанализированы ошибки классификации. 
Особое внимание уделялось балансу между точностью обнаружения спама и минимизацией ложных срабатываний, а также работе с дисбалансом классов и интерпретации полученных результатов и значимости признаков.

Для модели логистической регрессии был добавлен дополнительный признак — длина сообщения, что позволило существенно повысить итоговое значение метрики F1 score до 0.968.

В результате была получена модель, демонстрирующая устойчивое качество классификации и способность эффективно выявлять спам-сообщения. Также был подобран порог классификации, минимизирующий ложноположительные срабатывания.
В заключении проекта рассмотрены возможные направления для дальнейшего улучшения модели, включая добавление дополнительных признаков и использование более сложных методов представления текста и нейросетевых подходов.
